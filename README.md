# Support TG Bot

Телеграм-бот службы поддержки. Юзеры пишут боту с вопросами, 
сообщения автоматически попадают в супергруппу с топиками — 
на каждого юзера создаётся свой топик. Админы отвечают прямо 
из топика, ответ возвращается юзеру в личку.

> ⚠️ Проект в разработке. Часть функций ещё не реализована.

## Возможности

- 🎫 Создание тикетов через личку с ботом
- 💬 Двусторонняя переписка юзер ↔ админ
- 📁 Отдельный топик в супергруппе на каждого юзера
- 🖼 Пересылка сообщений любого типа (текст, фото, файлы и т.д.)
- 📝 Логирование событий

## Стек

- Python 3.12
- [aiogram](https://docs.aiogram.dev/) 3.x
- [aiosqlite](https://github.com/omnilib/aiosqlite) — async SQLite
- python-dotenv

## Запуск

1. Клонировать репозиторий:
```bash
   git clone https://github.com/DmitriProger/Support-tgbot.git
   cd Support-tgbot
```

2. Установить зависимости:
```bash
   uv sync
```

3. Создать файл `.env` в корне проекта:
```
   TOKEN=your_bot_token_here
   GROUP_ID=-100123456789
```

4. Запустить:
```bash
   python main.py
```

## Настройка группы

- Группа должна быть **супергруппой с включёнными топиками** (Forum)
- Бот должен быть **администратором** группы
- У бота должны быть права на **управление темами** (создание топиков)

## Структура проекта

```
app/
├── admin.py      — хендлеры для админов
├── user.py       — хендлеры для юзеров
├── topics.py     — работа с топиками супергруппы
├── keyboard.py   — инлайн-клавиатуры
└── states.py     — FSM состояния

database/
├── init.py       — инициализация БД
└── queries.py    — запросы к БД

main.py           — точка входа
```
